#  AI Language Translation Tool
A modern, user-friendly language translation application built with Python. Developed as part of the **CodeAlpha Artificial Intelligence Internship**.

##  Features
- **Real-time Translation:** Instantly translate text between multiple languages.
- **Modern UI:** Built using `CustomTkinter` for a sleek, dark-themed experience.
- **Powered by Google:** Uses the `deep-translator` library for accurate results.
- **One-Click Copy:** Easily copy the translated text to your clipboard.

##  Tech Stack
- **Language:** Python 3.x
- **GUI Library:** CustomTkinter
- **API:** Google Translate (via deep-translator)
- **Utilities:** Pyperclip (Clipboard management)
